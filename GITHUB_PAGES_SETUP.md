# Quick Setup Guide for GitHub Pages

## 🚀 Enable Your Documentation Website in 3 Steps

### Step 1: Go to Repository Settings
Visit: https://github.com/enyasystem/hotel-booking-app/settings

### Step 2: Enable GitHub Pages
1. Click **Pages** in the left sidebar
2. Under "Source", select:
   - **Branch**: `main`
   - **Folder**: `/docs`
3. Click **Save**

### Step 3: Wait for Deployment
- GitHub will begin building your site (usually 1-2 minutes)
- Once done, you'll see: "Your site is published at https://enyasystem.github.io/hotel-booking-app/"

## ❓ Troubleshooting

**Site not showing up?**
- Wait 2-3 minutes for deployment to complete
- Check that `/docs` folder exists
- Verify branch is set to `main`

**Images not loading?**
- Images are embedded from GitHub repository
- Verify `screenshots/` folder is committed
- Check image filenames are correct

**Need to disable?**
- Go back to Settings > Pages
- Select "None" under Source
- Click Save

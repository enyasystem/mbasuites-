## 📖 View the Guide

- **GitHub Pages** (once enabled): `https://enyasystem.github.io/hotel-booking-app/`

## 🚀 Enable GitHub Pages

To make this documentation publicly accessible via GitHub Pages:

1. Go to your repository: https://github.com/enyasystem/hotel-booking-app
2. Click on **Settings** (top right)
3. Scroll down to **Pages** (left sidebar)
4. Under "Source", select:
   - **Branch**: `main`
   - **Folder**: `/docs`
5. Click **Save**
6. GitHub will provide your public URL (usually takes 1-2 minutes to activate)

Your site will be available at: `https://enyasystem.github.io/hotel-booking-app/`

To update the documentation:

1. Edit `docs/admin-guide.html` or `docs/index.html`

Bookings Management

Manage reservations, confirm bookings, and track guest information efficiently.

Core Features

  • View all bookings with filtering by status
  • Search by guest name or booking ID
  • Confirm pending bookings instantly
  • Cancel reservations when needed
  • Track guest contact details and dates
Pro Tip: Always confirm pending bookings promptly to maintain clear communication with guests and prevent scheduling conflicts.

Rooms Management

Control your room inventory with complete CRUD operations and detailed management.

Creating a New Room

Provide the following information when adding a room:

  • Room Name – Descriptive name (e.g., "Luxury Suite")
  • Room Number – Unique identifier
  • Room Type – Standard, Deluxe, or Suite
  • Price Per Night – Base rate in your currency
  • Max Guests – Maximum occupancy
  • Description – Detailed features (minimum 10 characters)
  • Amenities – WiFi, AC, TV, Safe, etc.

Room Management

  • Edit existing room details anytime
  • Upload and manage room images
  • Toggle availability status
  • Delete rooms (only if no active bookings)
  • Set location assignments
Keep room numbers sequential and logical. Use high-quality images to increase booking rates.

Gallery & Room Media

Organize and manage all property photography and visual content.

Media Management

  • Upload multiple images per room
  • Drag-and-drop interface for easy organization
  • Set primary featured image
  • Automatic image optimization
  • Batch upload support
  • View and delete individual images
Image Guidelines: Use minimum 800px width. Include multiple angles, different lighting conditions, and highlight unique features. Professional photos increase conversion rates.

Hero Banner Management

Customize the main hero section displayed on your website homepage.

Customizable Elements

  • Title – Main headline (e.g., "Luxury Accommodations Await")
  • Subtitle – Secondary message (e.g., "Book Your Perfect Stay")
  • Background Image – Featured image (recommended: 1920×1080px)
  • Call-to-Action – Button text and destination

Best Practices

  • Keep headlines concise and compelling
  • Update seasonally for promotions
  • Use high-resolution images
  • Test on mobile devices before saving
  • Highlight unique selling propositions

Promotions Management

Create, manage, and track promotional campaigns and discount offers.

Promotion Types

  • Percentage-based discounts
  • Fixed amount discounts
  • Seasonal offers
  • Room-specific promotions

Creating Promotions

  • Define promotion code and name
  • Set discount type and value
  • Choose applicable date range
  • Select which rooms qualify
  • Set usage limits if needed
  • Include terms and conditions
Use memorable promotion codes. Monitor effectiveness and adjust rates based on conversion data. Blackout peak periods to protect revenue.

Calendar Synchronization

Sync your booking calendar with popular external calendar services.

Supported Integrations

  • Google Calendar
  • Microsoft Outlook
  • ICS Feed Import/Export
  • Generic Calendar Services

Setup Process

  1. Select calendar service to integrate
  2. Authenticate with your account
  3. Grant necessary permissions
  4. Configure sync preferences
  5. Test connection
  6. Activate sync
Test synchronization before going live. Monitor for sync errors and maintain calendar data consistency across systems.

Analytics & Reports

Gain insights into your business performance through comprehensive analytics.

Available Reports

  • Booking Analytics – Trends, patterns, and sources
  • Revenue Analytics – Income by period, room type, and location
  • Occupancy Analytics – Room utilization rates and trends
  • Guest Analytics – Demographics, behavior, and repeat rates
  • Operational Analytics – Staff, payment, and performance metrics

Key Metrics

  • Revenue Per Available Room (RevPAR)
  • Average Daily Rate (ADR)
  • Occupancy Rate
  • Booking patterns and seasonality
  • Guest satisfaction indicators
Review analytics weekly for actionable insights. Use data to inform pricing strategies, promotional campaigns, and inventory planning.

Staff Management

Control access and manage permissions for your admin and staff team members.

User Roles

  • Admin – Full access to all features and settings
  • Manager – Room and booking management capabilities
  • Staff – Limited access to assigned locations

Staff Operations

  • Add new team members with email invitation
  • Assign specific roles and permissions
  • Set location assignments
  • Deactivate inactive users
  • Reset forgotten passwords
  • Track staff activity in logs
Apply principle of least privilege – grant only the permissions each staff member needs. Review access regularly for security.

Activity Log

Comprehensive audit trail tracking all system activities and user actions.

Tracked Activities

  • User authentication (logins, registrations)
  • Booking actions (create, confirm, cancel)
  • Room management (add, edit, delete)
  • Profile modifications
  • Payment processing
  • Review submissions
  • Staff actions

Filtering & Search

  • Filter by entity type (Users, Bookings, Rooms, etc.)
  • Filter by action type (Create, Update, Delete)
  • Search by user name or email
  • Custom date range selection
  • Full-text search in action details
Review activity logs regularly for security and accountability. Investigate unusual patterns and maintain logs for compliance.

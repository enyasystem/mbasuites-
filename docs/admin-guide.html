Dashboard Overview

The Overview tab is your main dashboard that displays key metrics and recent activity.

Key Metrics

Active Bookings

Number of confirmed and pending bookings

Available Rooms

Count of rooms currently available for booking

Total Revenue

Cumulative revenue from all bookings

Occupancy Rate

Percentage of rooms occupied

Bookings Management

Manage all guest bookings, confirm reservations, and handle booking updates.

Key Actions

✅ Confirm Bookings

Change status from "Pending" to "Confirmed" and automatically send confirmation emails.

❌ Cancel Bookings

Mark bookings as cancelled and free up room availability.

📊 Filter & Search

Filter by status or search by guest name/booking ID.

Always confirm pending bookings promptly to maintain guest communication and prevent double-booking.

Rooms Management

Create, edit, and manage your hotel room inventory.

Required Information for New Rooms

  • Room Name - e.g., "Deluxe Suite"
  • Room Number - Unique identifier
  • Room Type - Standard, Deluxe, or Suite
  • Price Per Night - In your base currency
  • Max Guests - Maximum occupancy
  • Description - Detailed room features (min 10 chars)
  • Amenities - WiFi, AC, TV, Safe, etc.
Keep room numbers unique and logical. Use high-quality images for better bookings.

Gallery & Room Media

Manage and organize all room photographs and media.

Gallery Features

  • Upload multiple images per room
  • Drag-and-drop interface
  • Set primary images
  • Automatic image optimization
  • Batch uploads supported
Upload high-quality images (minimum 800px width). Include different angles and highlight unique features.

Hero Banner Management

Manage the main hero section on your website homepage.

Customizable Elements

  • Hero Title - Main headline
  • Subtitle - Secondary headline
  • Background Image - Featured hero image
  • Call-to-Action Button - Drives bookings
Keep headlines concise and impactful. Update seasonally for special promotions.

Promotions Management

Create and manage promotional offers and special deals.

Promotion Features

  • Create discount codes (percentage or fixed amount)
  • Set valid date ranges
  • Apply to specific rooms or all rooms
  • Set maximum uses
  • Track promotion performance
Use clear, memorable promotion codes. Blackout peak dates to protect revenue.

Calendar Sync

Synchronize your booking calendar with external calendars.

Supported Integrations

  • Google Calendar
  • Outlook Calendar
  • ICS Feeds
  • Other calendar services via ICS
Test sync before full deployment to prevent overselling and double-booking.

Analytics & Reports

Analyze business performance and generate insightful reports.

Available Reports

  • Booking Analytics - Trends and patterns
  • Revenue Analytics - Income by period/room
  • Occupancy Analytics - Room utilization rates
  • Guest Analytics - Demographics and behavior
  • Operational Analytics - Staff and payment metrics
Review analytics weekly for trends. Monitor occupancy rates and track RevPAR (Revenue Per Available Room).

Staff Management

Manage admin and staff team members.

Staff Roles

  • Admin - Full access to all features
  • Manager - Management of rooms and bookings
  • Staff - Limited access to assigned locations
Grant minimal permissions needed for each role. Regularly review staff access permissions.

Activity Log

View comprehensive audit trail of all system activities.

Tracked Activities

  • User logins and registrations
  • Booking actions (create, confirm, cancel)
  • Room management (add, edit, delete)
  • Profile and settings changes
  • Payment transactions
  • Review submissions

Filtering Options

  • Filter by entity type (Users, Bookings, Rooms, etc.)
  • Filter by action (Create, Update, Delete, Confirm)
  • Search by user name or email
  • Custom date ranges
Review activity logs weekly for accountability and to investigate unusual patterns.

Payment Management

Manage guest payments and financial transactions.

Payment Operations

  • View pending payment requests
  • Request payments from guests
  • Process payments and upload proof
  • Extend payment deadlines
  • Send payment reminders
Keep payment deadlines clear and send reminders regularly. Verify all proof documents.

Guest Management

Manage guest registrations and information.

Guest Features

  • Guest Profiles - Full contact and preference information
  • Booking History - Track all guest reservations
  • Preferences - Room type, special requests, dietary needs
  • VIP Flags - Identify and reward loyal guests
  • Guest List - Browse all registered guests
Keep guest information updated. Use preferences for personalization and segment guests for marketing.

Payment Settings

Configure payment methods and financial settings.

Configuration Options

  • Payment Methods - Enable/disable accepted methods
  • Currency Settings - Primary currency and conversion
  • Payment Terms - Policies and deadlines
  • Banking Information - Account and recipient details
  • Payment Gateway - Processor and API setup
Use multiple payment options for guest convenience. Test payments regularly and maintain PCI compliance.
